# FlexPrint Pune - Premium Flex Printing Services

A modern React application for FlexPrint Pune, a professional flex printing press based in Pune, India.

## Features

- 🖨️ **Interactive Price Calculator** - Get instant quotes for flex printing orders
- 📱 **Fully Responsive** - Works seamlessly on desktop, tablet, and mobile
- 🎨 **Modern UI** - Sleek, professional design with custom SVG illustrations
- 🔄 **Hash-based Routing** - Simple navigation without external dependencies
- ⚡ **Fast Performance** - Optimized with code splitting and lazy loading

## Tech Stack

- **React 18** - UI library
- **Tailwind CSS** - Styling
- **Custom Hooks** - For routing and calculations
- **PropTypes** - Type checking
- **Classnames** - Conditional class management
